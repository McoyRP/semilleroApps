Prueba de hipotesis para la media $\mu$
=======================

Suponga que se tiene una muestra aleatoria $$x_1, x_2, \ldots, x_n$$ proveniente de una poblacion normal. Se quiere estudiar la hipotesis nula $H_0: \mu = \mu_0$ y se sospecha que:

1. $H_A: \mu < \mu_0$
2. $H_A: \mu \neq \mu_0$
3. $H_A: \mu > \mu_0$
